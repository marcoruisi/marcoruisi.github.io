Colophon

Every project says something through the way it’s built.

I wanted this website to be no exception.

It’s intentionally simple.

Few dependencies.

No unnecessary elements.

The same principles I try to apply when designing systems guided the creation of this site.

Reuse before replacing.

Add only when it truly adds value.

Prefer simple, open, and understandable tools.

Let every component do its job well.

The site is hosted on GitHub Pages.

It’s built with plain HTML, CSS, and minimal JavaScript, without a CMS, database, or frontend framework.

The content is written directly in HTML, prioritizing simplicity, readability, and long-term maintainability.

Images are kept to a minimum.

Dependencies even more so.

This project grew out of many conversations with ChatGPT.

Not to have a website written for me.

But to help me find the right words for a way of thinking that had been with me for years, yet that I had never really put into writing.

I tried to build this website in the same way I like to design systems: simple, understandable, and built to last.
